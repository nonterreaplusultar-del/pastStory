# 主题矩阵

| 主题 | 出现在哪些文章 | 表层含义 | 深层含义 | 是否有变化 |
|---|---|---|---|---|
| 爱情 | 《西安爱情故事》《无法忘记的人》《前几日夜梦所感》《最速小丑记》《致未来爱人的信》《梦见师兄结婚》《夜未眠所想》《生活一些琐碎》《为什么两次有这么大的差别》《早上的梦》《一些无病呻吟》《白发女的故事》《反思》《感想》《写给未来》 | 回忆初恋、后悔、渴望爱情 | **用"深情"来构建自我身份**。爱情叙事是他的核心自我认知方式——他不是在写爱情，是在用爱情定义自己是谁 | 有：从"想复合"→"梦见愤怒"→"叙事闭合"，方向是从执念走向完成 |
| 西安 | 《西安爱情故事》《无法忘记的人》《写给未来》《一些回忆》《梦中记》 | 青少年时期的城市背景 | **失去的坐标系**。西安不是一座城市，是他失去的一切的代名词 | 没有实质性变化，作为背景持续出现 |
| 青少年时期 | 《西安爱情故事》《为什么两次有这么大的差别》《不要苛责年轻的自己》 | 不成熟的自己、当年的错误 | **自我审判的法庭**。他对青少年自己的评价是持续进行的审判程序 | 有：从"全是我的错"到"当年的自己也很难" |
| 出国/远方 | 《启程新加坡》《新加坡初记》《告别新加坡》《记录的意义》 | 出国交换、异国体验 | **救赎期待的落空**。远方被想象成解决方案，但实际没有解决问题 | 有：从"新的开始"到"没有什么值得留恋" |
| 漂泊 | 《新加坡初记》《告别新加坡》《写给未来》《9月杂谈》《12月杂记》《近期日常》 | 在不同地方之间移动 | **精神上的无处安放**。他换了很多地方但没换掉焦虑 | 有：从地理意义上的漂泊到对"安顿"本身的渴望 |
| 自我证明 | 《写给未来》《刷完HOT100》《近日情况》《为什么不推荐读博士》《11月起居注》《12月杂记》 | 刷题、保研、学历 | **安全感的替代品**。他认为成功会治愈一切——包括失去的爱情 | 有：从"我要一飞冲天"到"我只想拿个offer"——期待值下降 |
| 与同龄人的比较 | 《11月杂记（2024）》《近日情况》《看见师兄结婚有感》《读知乎收藏有感》 | 别人有女友/offer/婚姻，我没有 | **自卑和自傲的双重奏**。他既觉得别人不行，又觉得自己不如别人 | 没有实质性变化，持续的比较焦虑 |
| 焦虑 | 几乎全部文章 | 对未来不确定性的恐惧 | **一种身份**而非一种情绪。他把焦虑写成了自己的定义性特征 | 有：对象从"过去"转向"未来" |
| 后悔 | 《无法忘记的人》《前几日夜梦所感》《最速小丑记》《夜未眠所想》《反思》《感想》 | 对初恋处理方式的后悔 | **道德赎罪券**。后悔是一种仪式：重复后悔=我已经付了代价 | 有：从"全是我的错"到更复杂的归因 |
| 孤独 | 《新加坡初记》《告别新加坡》《9月杂谈》《11月起居注》《12月杂记》《健身困难》 | 缺乏深层连接 | 不是没人，是**无法真正和人在深层建立连接** | 有：从"被环境孤立"到"可能是我自己的问题" |
| 自尊 | 《前几日夜梦所感》《夜未眠所想》《白发女的故事》《11月杂记（2024）》《龙年第一次失败》 | 被拒绝后的反应、不愿低头 | **脆弱的自我保护**。他的高度自尊掩盖的是对被否定的极端敏感 | 没有实质性变化，是核心人格特征 |
| 自恋 | 最难看到的部分，但存在于很多文章的叙事姿态中 | 把自己写成主角、悲剧性人物 | **痛苦即特殊**的潜意识。他觉得自己的痛苦使他区别于普通人 | 有微弱变化：2026年的《白发女的故事》中锐评部分来自外部视角 |
| 逃避 | 《告别新加坡》《为什么不推荐读博士》《11月起居注》《一些无病呻吟》 | 不想读博是因为"不喜欢学术" | 把恐惧包装成选择。他说"我不想"的时候往往意思是"我怕做不到" | 有：2026年开始更直接地承认"因为没找到工作才读博" |
| 成长 | 《不要苛责年轻的自己》《写给未来》《白发女的故事》后记、《12月杂记》年终总结 | 从经历中学习 | **认知层面的成长快于行为层面**。他知道了对错但还没改掉习惯 | 有：真实的认知升级，但行为改变滞后 |
| 写作本身 | 《记录的意义》《11月起居注》《写给未来》 | 记录是为了不遗忘 | **写作是他的自我疗愈工具，也是他的逃避工具**。他通过写作来确认自己的感受是真实的、重要的 | 有：从"记录大事"到"记录日常"，姿态更放松 |
